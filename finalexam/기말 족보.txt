5. 다음 배열 a에서 포인터 p, q를 선언해 각각 배열의 가장 앞과 뒤의 주소를 저장한 후, 배열 a의 순서를 역으로 바꾸어 출력하는 완전한 프로그램을 작성하시오.
int a[] = {11,22,33,44,55,66};
출력결과는 "김현진"으로 대치해야 하고, 중요한 논리를 설명하는 주석을 적절하게 포함시켜야 하며, 변수명도 평가의 대상이 된다.
표절 의심될 시 무조건 0점 처리.
-----
박혜자씨 출력 결과는 다음과 같습니다.
66 55 44 33 22 11
-----
내가 쓴 코드는 아래와 같음
#include <stdio.h>

int main(void)
{
    int a[] = { 11, 22, 33, 44, 55, 66 }; // 주어진 배열
    int size = sizeof(a) / sizeof(a[0]); // 배열 크기

    int* p = a; // 첫 번째 주소
    int* q = &a[size - 1]; // 배열 마지막 주소

    printf("김현진씨 출력 결과는 다음과 같습니다.\n"); // 내 이름

    // a의 순서를 역으로
    while (q >= p) {
        printf("%d ", *q); // 현재 가르키는 값 출력 후
        q--; // 전으로 이동
    }
    printf("\n");

    return 0;
}
